## Installation

```
yarn add lazyloadsrc
```

## Usage

### HTML

```
<img src='' data-src='/path/to/img' />
```

### TypeScript / JavaScript

```
import LazyLoadSrc from 'lazyloadsrc'
const lazyloadsrc = new LazyLoadSrc()
```
